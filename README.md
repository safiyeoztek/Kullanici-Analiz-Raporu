# Kullanıcı Analiz Raporu
### Bu Python uygulaması, kullanıcıdan aldığı dinamik verileri işleyerek kişiselleştirilmiş bir analiz raporu sunar. Metin işleme teknikleri ve mantıksal denetimler üzerine odaklanmıştır.

## Ne Yapıyor?
* input() fonksiyonu ile kullanıcıdan isim, yaş ve teknoloji tercihlerini alır.

* İsim ve soyisimi birleştirerek tamamen büyük harfe (upper()) çevirir.

* Girilen metinlerin uzunluğunu hesaplar.

* Kullanıcının girdiği teknolojileri bir listeye dönüştürür (split()).

* Yaş ve teknoloji sayısı kriterlerine göre kullanıcıya özel geri bildirimler verir.

## Hangi Konular Var?
* Kullanıcı Etkileşimi: input() ve int() tip dönüşümü.

* String Metotları: .upper(), .split() ve len() fonksiyonu.

* Veri Yapıları: Listeler (List).

* Kontrol Akışı: if-else yapısı ile mantıksal sorgulama.

## Nasıl Çalıştırılır?
* Bilgisayarınızda Python yüklü olduğundan emin olun.

* Terminalde proje klasörüne gidin.

* Aşağıdaki komutu çalıştırın:

```Bash
python analiz_raporu.py

```
* Ekrandaki talimatları izleyerek bilgilerinizi girin.
